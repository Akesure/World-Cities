# World Cities List

Multilanguage list of countries, states & cities in XML format.

## Language support

* Chinese Simplified: zh-cn.xml
* Chinese Traditional: zh-tw.xml
* German: de.xml
* English: en.xml
* Spanish: es.xml
* French: fr.xml
* Japan: jp.xml
* Korean: ko.xml

## Data Source

* QQ and QQ Int.
